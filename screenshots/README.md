This folder will contain screenshots for the README.

To generate screenshots:

1. Run the example project: cd example_project && python manage.py runserver
2. Visit http://localhost:8000/all-features/
3. Take screenshots of each demo section
4. Save with these filenames:
   - light-theme.png
   - dark-theme.png
   - dashboard.png
   - all-features.png
   - color-schemes.png
